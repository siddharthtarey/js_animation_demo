List of Files:

	- index.html : the main file, that contains the UI(this file should be opened in the browser)
	- tabledata.js : the js file that creates the table 
	- animation.js : contains the fade animations required for this project.
	- style.css : contains the styles for the HTML page
	- browser.html : the html file that loads if browser is lower than IE 7.
	- data_new.json : The data source file.
	- /images folder : Contains background images and browser logos for browser.html
	
How to use:

	- The drop-down on the left wil populate genres,sub-genres and artist dynamically
	- The list on the right are the songs related to that genre or artist.
	- The song list dynamically populates based on the selections made in the drop down.
